# PNG切分工具 - 纯JS版本

这是一个简单的PNG图片切分工具，**不需要任何框架或构建工具**，可以直接在浏览器中打开使用。

## 功能说明

这个工具可以：
- 自动检测PNG图片中的Alpha通道（透明区域）
- 识别并切分所有独立的UI组件
- 每个组件会被切分成独立的PNG图片
- 支持预览原始图片和切分结果
- 支持单独下载或批量下载切分后的图片
- **支持多语言切换**：中文、英文、日文、法语、西班牙语、韩语

## 使用方法

1. **直接打开**：双击 `index.html` 文件，在浏览器中打开
2. **或者**：在浏览器中打开 `file:///你的路径/simple-js/index.html`

## 文件说明

- `index.html` - 主页面，包含HTML结构
- `style.css` - 样式文件，包含所有CSS样式
- `app.js` - 主应用逻辑，处理文件上传、UI更新和多语言切换
- `imageProcessor.js` - 图片处理核心逻辑，包含Alpha通道检测和切分算法
- `locales.js` - 多语言翻译文件，包含6种语言的翻译文本

## 多语言支持

页面支持以下6种语言，可在页面右上角切换：

- 🇨🇳 **中文（简体）** - 默认语言
- 🇺🇸 **English（英语）**
- 🇯🇵 **日本語（日语）**
- 🇫🇷 **Français（法语）**
- 🇪🇸 **Español（西班牙语）**
- 🇰🇷 **한국어（韩语）**

**语言偏好记忆**：选择的语言会自动保存到浏览器本地存储，下次打开页面时会自动使用上次选择的语言。

## 技术说明

### 核心算法

1. **Alpha通道检测**：读取PNG图片的每个像素的Alpha值，识别非透明区域
2. **连通区域检测**：使用深度优先搜索（DFS）算法，找到所有独立的连通区域
3. **矩形切分**：为每个连通区域计算最小外接矩形，并切分成独立的图片

### 代码注释

所有代码都包含详细的中文注释，解释了：
- 每个函数的作用
- 算法的实现原理
- 关键步骤的处理逻辑

## 注意事项

- 只支持PNG格式的图片
- 需要图片包含Alpha通道（透明区域）
- 太小的区域（少于10个像素）会被过滤掉，避免噪点干扰
